foo_cad_plus

Enhanced CD Art Display/Rainmeter/SAO Utils interface plugin for foobar2000.

GitHub project: RangerCD/foo-cad-plus

Installation

  • Ensure that foobar2000 1.1.11 or higher is installed
  • Download foo_cad_plus 1.3.2
  • Run (double-click) the downloaded file to install

Note 1: If you are using foo_cad, make sure to REMOVE foo_cad before installing foo_cad_plus, use Library → Configure → Components, right click CD Art Display and select Remove. All features are reserved and 100% compatible in foo_cad_plus.

Note 2: If you are running a portable version of foobar2000, use Library → Configure → Components → Install.... Then, select the downloaded foo_cad.fb2k-component file to install.
